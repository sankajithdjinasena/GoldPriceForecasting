# 📈 Gold Price Time Series Dataset (2014--2026)

## 📌 Overview

This dataset contains **daily gold prices** from **January 1, 2014 to
January 2, 2026**.

The data is pre-processed and structured specifically for:

-   📊 Time Series Analysis
-   🔮 Price Forecasting
-   🤖 Machine Learning Models
-   📉 Trend & Pattern Analysis

All prices are recorded in **INR (Indian Rupee)** for **10 grams of 24K
gold**.

------------------------------------------------------------------------

## 📂 Dataset Details

-   **Time Period:** 2014-01-01 to 2026-01-02
-   **Frequency:** Daily
-   **Currency:** Indian Rupee (INR)
-   **Unit:** 10 grams (24K Gold)
-   **Format:** CSV (Pre-processed for time series analysis)

The dataset is structured to be directly usable for forecasting models
such as:

-   ARIMA / SARIMA
-   LSTM / RNN
-   Prophet
-   Regression-based models
-   Deep Learning architectures
